## モジュール間依存関係

```plantuml
@startuml
skinparam backgroundColor #FAFAFA
left to right direction

title モジュール間依存関係図

package "EditorQt" {
  [MainWindow]
  [EditorRuntimeHost]
  [RuntimeBridge]
  [QtEditorFrontend]
  [ImGuiEditorFrontend]
}

package "ApplicationDLL" {
  [AppRuntime] as AR
  [FrameLoop] as FL
  [PlayInEditor] as PIE
  [EditorUi] as EU
  [Dx12RenderDevice] as DX12
  [SpriteRenderer系] as SR
  [TextureAssetManager] as TAM
  [PMDAnalyzer] as PMD
}

package "PieGameManaged" {
  [GameEntry] as GE
  [Scene] as SCN
  [GameObject] as GO
  [SpriteRendererSystem] as SRS
  [NativeMethods] as NM
}

[MainWindow] --> [EditorRuntimeHost]
[EditorRuntimeHost] --> [RuntimeBridge]
[RuntimeBridge] --> AR : dllimport
[MainWindow] --> [QtEditorFrontend]
[QtEditorFrontend] --> [RuntimeBridge]

AR --> TAM

GE --> SCN
SCN --> GO
GO --> SRS
SRS --> NM

@enduml
```

## レンダリングバックエンド切り替え

実行時に `SetRendererBackend(backend)` を呼ぶことで切り替え可能。

```plantuml
@startuml
skinparam backgroundColor #FAFAFA

state "レンダリングバックエンド選択" as choice <<choice>>

[*] --> choice : SetRendererBackend(n)

choice --> DirectX12 : n=0\n(既定)
choice --> Vulkan : n=1
choice --> OpenGL : n=2

state DirectX12 {
  Dx12RenderDevice --> Dx12SpriteRendererBackend
  Dx12RenderDevice --> DescriptorHeapManager
  Dx12RenderDevice --> PipelineLibrary
  Dx12RenderDevice --> RootSignatureCache
}

state Vulkan {
  VulkanRenderDevice --> VulkanSpriteRendererBackend
}

state OpenGL {
  OpenGLRenderDevice --> OpenGlSpriteRendererBackend
}

@enduml
```

## フレームループ実行順序

`FrameLoop` クラスがフレームごとの処理順を定義する。

```plantuml
@startuml
skinparam backgroundColor #FAFAFA
skinparam activityBackgroundColor #E3F2FD
skinparam activityBorderColor #1565C0

title フレームループ実行順序

start

:Win32 メッセージ処理;
note right: PeekMessage / TranslateMessage

if (PIE 実行中?) then (yes)
  :PlayInEditor::Update()\n→ GameEntry::GameTick(delta);
  :SpriteRendererSystem::Sync()\n(C#→ネイティブ状態同期);
endif

:IRenderDevice::PreRender(clearColor);
note right: バックバッファクリア

if (EditorUi 有効?) then (yes)
  :EditorUi::RenderFrame()\n(ImGui ウィジェット描画);
endif

:IRenderDevice::Render();
note right: スプライト・メッシュ描画\nPresent

stop

@enduml
```
